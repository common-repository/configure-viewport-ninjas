Need to fix your Configure Viewport error on Google PageSpeed? This allows you to choose between responsive and fixed width.

== Description ==

Deceptively simple plugin allows you to choose between a responsive or a fixed width mobile view - just like Google PageSpeed Insights asks you to do. Install the plugin, navigate to the menu, and select your options. It is that easy.

== Installation ==

1. Upload '/configure-viewport-ninjas/` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Navigate to "Configure Viewport" on wp-admin left side menu. Select your options for mobile display.
